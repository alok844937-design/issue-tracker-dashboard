# Open Source Issue Tracker Dashboard 
A simple web tool to explore open-source GitHub issues, especially begineer-friendly ones like `good first issue`.

## Why this project?
Finding the right issue to start contributing to open source is difficult for begineers. This dashboard simplifies that process.

## Features 
- Fetch open issues from any public GitHub repository
- Filter `good first issue` labels
- Direct links to GitHub issues
- Deployed using GitHub Pages

## Tech Stack 
- HTML 
- CSS
- JavaScript
- GitHub REST API
